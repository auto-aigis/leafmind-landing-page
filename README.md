Submit email to `https://leafmind-landing-page.onrender.com?source=twitter`\n2. Verify success message appears\n3. Check Supabase `waitlist` table:\n   - Record exists\n   - source = 'twitter'\n   - submitted_at is recent\n\nSee `SMOKE_TEST.md` for detailed procedure.\n\n---\n\n## 🛠 Troubleshooting\n\n### Form won't submit\n- Check browser console for errors\n- Verify Supabase credentials\n- Check API route logs: `frontend/app/api/waitlist/route.ts`\n\n### Can't login\n- Verify email_verified flag in users table\n- Check password_hash is set\n- Try registration first if new user\n\n### Analytics not showing\n- Verify NEXT_PUBLIC_VERCEL_ANALYTICS_ID is set\n- Allow 5 minutes for data ingestion\n- Check Vercel project settings\n\n---\n\n## 📚 Documentation\n\n- **DEPLOYMENT.md** - Step-by-step production setup\n- **IMPLEMENTATION.md** - Full technical overview\n- **SMOKE_TEST.md** - Go/no-go verification checklist\n\n---\n\n## 📞 Support\n\nFor issues:\n1.
